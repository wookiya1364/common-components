# Commit Messages Style

|타입|설명|
|-----|-----|
|feat|새로운 기능 추가|
|fix|버그 수정|
|docs|문서 수정|
|refactor|코드 리펙토링|
|test|테스트 코드 추가|
|perf|성능 개선|
|design|디자인 변경|
